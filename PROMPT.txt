# CONTEXTO DO PROJETO

Eu estou participando de um hackathon chamado "Cidadão na Rede - FIB16", que ocorre em Belém/PA. Meu projeto se chama "Guardião da Rede" e tem como tema:

- Segurança digital
- Uso responsável e deveres na Internet

## Problema que resolvemos
Os vídeos do Cidadão na Rede são excelentes para conscientização geral, mas não oferecem suporte personalizado ou baseado na experiência da comunidade. Vítimas de golpes virtuais não têm um canal anônimo e inteligente para verificar se uma situação suspeita é realmente um golpe.

## Nossa solução (MVP)
Um site responsivo com:
1. Formulário anônimo para relato de suspeita de golpe (título + descrição + imagem opcional)
2. Triagem sociodemográfica opcional (similar ao Akinator - perguntas curtas sobre perfil da vítima)
3. Motor de análise baseado em similaridade com relatos anteriores + diretrizes oficiais
4. Resposta automática com nível de risco (baixo/médio/alto) e recomendações
5. Hub público com categorias de golpes mais reportados
6. Chatbot no Telegram (versão simplificada) para relatos rápidos

## Público-alvo
- Pessoas leigas em tecnologia
- Vítimas potenciais de golpes (idosos, pequenos empresários, MEIs, usuários comuns)
- Faixa etária indefinida (acessível para todos)

# OBTENÇÃO DE API KEY (COTA GRATUITA)

Você precisará usar uma API de IA em algum momento (ex: para refinar as respostas ou gerar recomendações personalizadas). Siga as instruções abaixo, por ordem de preferência:

## Opção 1: DeepSeek (via API oficial - mais barata do mercado)
1. Acesse https://platform.deepseek.com/
2. Crie uma conta (pode ser com e-mail do Google)
3. Vá em "Chaves de API" no menu lateral esquerdo
4. Clique em "Criar nova chave de API"
5. Dê um nome (ex: "hackathon-guardiao")
6. Copie a chave gerada (será algo como sk-...)
7. Para usar a API, você precisa comprar créditos (é pré-pago):
   - O preço é ~R$ 0,14 por milhão de tokens de entrada e ~R$ 0,55 por milhão de tokens de saída
   - Para teste, basta depositar o valor mínimo (geralmente US$ 5 ou equivalente)
   - **Ou**, se quiser testar sem pagar agora, use o chat web do DeepSeek gratuitamente (mas aí não será via API)

## Opção 2: Google Gemini (com AI Studio gratuito - sem custo inicial)
1. Acesse https://aistudio.google.com/
2. Faça login com sua conta Google
3. Clique em "Get API key" no menu lateral
4. Clique em "Create API key"
5. Escolha "Gemini API" (a chave será gerada automaticamente)
6. Copie a chave e guarde

✅ **O Gemini tem uma cota gratuita generosa** (60 requisições por minuto, sem custo por vários meses para uso moderado). Esta é a opção mais recomendada para o hackathon, pois você não precisa colocar cartão de crédito.

## Como usar a chave no Antigravity/Projeto:
- Se usar **Gemini**: instale o SDK com `npm install @google/generative-ai` (JavaScript) ou `pip install google-generativeai` (Python)
- Se usar **DeepSeek**: a API é compatível com OpenAI, então use `npm install openai` e mude a `baseURL` para `https://api.deepseek.com`

## Instrução final sobre API:
**Por favor, escreva no seu código um arquivo `.env.example` mostrando as variáveis necessárias (ex: GEMINI_API_KEY, DEEPSEEK_API_KEY), e inclua comentários explicando como obter cada chave. Não coloque chaves reais no código.**
